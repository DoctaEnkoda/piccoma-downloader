# Piccoma-downloader
Download manga from [Piccoma](https://piccoma.com/)

## Usage
Make sure you have a [node.js](https://nodejs.org/) environment
1. Clone or download this repository
2. Run `npm install`
3. Run `node piccoma.mjs`

![usage](usage.png)
- compress downloaded mangas into zip
```
node archive.mjs 
```
